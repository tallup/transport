# User guide screenshots

Optional images for the in-app **Parent**, **Admin**, and **Driver** help pages. The React components reference files here with URLs like `/images/user-guides/parent/01-dashboard.png` (served from `public/`).

## Layout

- `parent/` — parent portal screenshots
- `admin/` — admin console screenshots
- `driver/` — driver portal screenshots

## Naming

Use a numeric prefix so files sort in reading order, then a short slug:

- `01-dashboard.png`
- `02-bookings.png`

Use lowercase and hyphens in filenames. Prefer **PNG** or **WebP**; compress for web before committing.

## Size and content

- Recommended capture width: **1280px** (consistent across all shots).
- Avoid real personal data—use demo accounts, blur, or crop sensitive fields.
- Every image should have a meaningful `alt` (caption) in the guide page.

Images are **optional** until your team captures them; missing files show a dashed placeholder with the expected path.
